## Project Overview
This project involves building a small website foundation for Kauai, a South African health food restaurant chain, for the WEDE5020 Part 1 assignment. The assignment focuses on establishing structural, semantic HTML pages along with planning, requirement gathering, and basic setup for future CSS and JavaScript components. 

## Website Goals and Objectives
- **Increase brand awareness:** Present Kauai's story, values, and product range to new and existing customers through an engaging, informative online presence.
- **Showcase the menu:** Display Kauai's full product range including meals, wraps, smoothies, and seasonal specials.
- **Drive foot traffic:** Use store location pages and an embedded map to direct customers to their nearest Kauai outlet.
- **Generate franchise and catering enquiries:** Provide a structured enquiry form for potential franchisees and corporate catering clients.
- **Support brand loyalty:** Communicate promotions, health tips, and rewards programme information to returning visitors.

## Key Features and Functionality
- **Multi-page navigation:** Full site linking across five structural pages (Home, About Us, Menu, Enquiry, Contact).
- **Semantics:** Full usage of standard HTML5 elements including `header`, `nav`, `main`, `section`, `footer` and `form`.
- **Enquiry Form:** Form for capturing detailed business queries like catering and franchising.
- **Multiple Location Embeds:** Contact page features Google Maps embeds showcasing locations in Cape Town, Johannesburg, and Gqeberha.

## Part 1 Details
In Part 1, the core functionality delivered includes:
- Thorough research mapped into a detailed Word Document Proposal.
- Semantic HTML tags have been correctly utilized across all pages.
- Correctly defined parent and child hierarchical linking for five key website sections.
- Creation of placeholders within the code utilizing inline comment methodology to establish best coding practices before jumping to CSS configuration.
- Setup of organized local directories specifically containing index.html, about.html, service.html, inquiry.html, contacts.html alongside placeholder folders (css, js, images).
